# EduShare – CW2 Cloud Native Implementation

This repository contains the implementation of the EduShare multimedia sharing platform
developed for Coursework 2 of COM682 – Cloud Native Development.

The solution implements a cloud-native architecture using Microsoft Azure services,
including Azure Blob Storage, Azure Cosmos DB, Azure Functions, Logic Apps,
Application Insights, and CI/CD using GitHub.

The implementation is based on the system design produced in Coursework 1.
